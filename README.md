# 📊 Nassau Candy Profitability & Business Intelligence Analysis

An end-to-end **data analytics & business intelligence project** focused on analyzing product profitability, cost efficiency, and division performance for Nassau Candy Distributor.

This project transforms raw transactional sales data into actionable insights using Python analytics and an interactive Streamlit dashboard to support data-driven business decision-making.

---

## 🚀 Project Objective
The goal of this project is to:

- Identify high-profit and low-profit products  
- Detect margin inefficiencies and cost-heavy items  
- Analyze revenue vs profit imbalance across divisions  
- Perform profit concentration (Pareto) analysis  
- Build KPI-driven insights for strategic decisions  
- Develop a live interactive dashboard using Streamlit  

This simulates a real-world business analytics case used by management teams for pricing, cost optimization, and product strategy.
